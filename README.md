### How to run locally

This app requires [Node.js](https://nodejs.org/) v6.4+ to run.

```sh
$ git clone https://github.com/ben198/weather-app.git
$ cd weather-app
$ npm install
```

Get your free API key from [Darksky](https://darksky.net) and include it in the config.js file at the root of the project.

```
$ npm start
```

Open your web browser and open http://localhost:8080/ in a new tab
